# Design System - WellPlace

## Visual Direction

- Paleta: Verde moderno
- Estilo: Moderno/Bold
- Layout: Sidebar collapsible + top navbar

## Paleta (HSL)

- Primary: `--primary: 152 55% 36%`
- Secondary: `--secondary: 152 32% 90%`
- Accent: `--accent: 168 52% 48%`
- Background: `--background: 36 40% 98%`
- Foreground: `--foreground: 166 28% 12%`
- Border: `--border: 150 20% 84%`

## Tipografia

- Font principal: Space Grotesk
- Peso destacado: 600+ para titulos
- Cuerpo: 400-500
